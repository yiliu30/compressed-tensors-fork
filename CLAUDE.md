# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Build & Test Commands

- **Format**: `make style` (runs copyright fix + black + isort)
- **Lint/Quality**: `make quality` (copyright check + black --check + isort --check + flake8)
- **Test all**: `make test` (runs `pytest -ra tests`)
- **Single test**: `pytest -ra tests/path/to/test_file.py -k 'test_name'`
- **Install dev**: `pip install .[dev]`
- **Build**: `BUILD_TYPE=release make build`

## Code Style

- Black with `line-length=88`, target `py310`
- isort with `profile=black`
- Flake8 ignores: E203, E251, E701, W503
- Every `.py` file must begin with SPDX header:
  ```python
  # SPDX-License-Identifier: Apache-2.0
  # SPDX-FileCopyrightText: Copyright contributors to the vLLM project
  ```

## Project Layout

- `src/` layout: package at `src/compressed_tensors/`
- `version.py` is auto-generated by setuptools_scm — do not edit
- Versioning uses git tags + `BUILD_TYPE` env var (release/nightly/dev)
- Test structure mirrors src: `tests/test_<module>/`

## Test Conventions

- Pytest markers: `unit`, `smoke`, `sanity`, `regression`, `integration`
- `@requires_gpu` / `@requires_gpu(N)` skips tests without GPU
- `@torchrun(world_size)` spawns distributed test subprocesses
- GPU tests run on `gcp-k8s-vllm-l4-duo` in CI

## Device API

- Uses `torch.accelerator` API (not hard-coded CUDA) — device-agnostic for XPU/CUDA/etc.
- Minimum PyTorch version: `torch>=2.10.0`
- Helper: `is_accelerator_type()` in `src/compressed_tensors/utils/helpers.py`

## Branch & Commit Conventions

- Branch prefixes: `feat/`, `fix/`, `chore/`, `refactor/`, `test/`, `docs/`
- Conventional commit messages (e.g., `feat: add X`, `fix: resolve Y`)
- Include `Signed-off-by` tag in commits
- Mergify auto-labels PRs needing rebase or failing quality checks
